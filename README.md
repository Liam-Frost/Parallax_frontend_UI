# Frontend_Test

一个使用原生 HTML/CSS/JavaScript 编写的简易前端示例，演示了用户注册/登录与车牌登记的基本流程。所有数据都保存在浏览器 `localStorage` 中，方便在本地快速验证交互逻辑。

## 功能

- **用户注册**：输入用户名和密码创建账户，同步执行简单的长度和重复校验。
- **用户登录**：使用注册信息登录，并在刷新页面后依旧保持登录状态（基于本地会话存储）。
- **车牌登记**：登录后可以登记车牌号及车辆型号，校验重复提交并支持删除。登记数据仅对当前用户可见。

## 使用方法

1. 使用任意静态文件服务器或直接在浏览器中打开 `index.html`。
2. 在“注册”标签页中创建新用户，或切换到“登录”标签页使用已有账号登录。
3. 登录成功后填写车牌号与车辆型号完成登记，可在列表中查看、删除已登记的记录。

> 说明：本示例仅用于前端交互演示，所有数据存放在浏览器本地，未与任何后端接口交互。
